## 動作環境
・RealTrainMod1.12.2 or KaizPatch

## 使用方法
1.モデルを組み込む。
2.テクスチャ・スクリプトを配置する。
4.jsonで材質をスクリプトのパスを指定する。
["K-ATS", textures/train/K-ATS/ATS_PANEL.png", "AlphaBlend,Light"]],
"rendererPath": "scripts/Render_Kamikawa_ATS.js"},
